Here's a prompt that should work well with AI UI design tools like Uizard, Visily, or Galileo AI.

---

**Prompt:**

> Design a modern, premium desktop application for a touchless AI-powered virtual mouse. The application uses MediaPipe hand tracking through a webcam to control the computer cursor without touching a physical mouse.
>
> The design should be clean, futuristic, and professional, similar to applications from Microsoft, Adobe, or Logitech. Use a minimal interface with rounded corners, subtle shadows, glassmorphism effects where appropriate, smooth animations, and a dark theme with blue and cyan accent colors.
>
> The application name is **AirMouse AI**.
>
> ### Main Layout
>
> * Left collapsible navigation sidebar with icons and labels.
> * Top header containing application title, webcam status, AI status, FPS indicator, and Settings button.
> * Main dashboard with responsive cards and organized sections.
>
> ### Dashboard Sections
>
> **1. Live Camera Preview**
>
> * Large webcam preview showing detected hands.
> * Hand landmarks overlay.
> * Status indicator (Hand Detected / No Hand Detected).
> * FPS counter.
>
> **2. Cursor Control**
>
> * Enable/Disable virtual mouse.
> * Cursor sensitivity slider.
> * Cursor smoothing slider.
> * Pointer speed adjustment.
> * Screen mapping visualization.
>
> **3. Gesture Controls**
> Display gesture cards with icons:
>
> * Move Cursor
> * Left Click
> * Right Click
> * Double Click
> * Drag & Drop
> * Scroll
> * Volume Control
> * Brightness Control
> * Copy
> * Paste
> * Cut
> * Undo
> * Redo
> * Next
> * Previous
> * Play/Pause
>
> Each gesture card should show:
>
> * Gesture icon
> * Gesture name
> * Assigned finger pose
> * Enable toggle
> * Edit button
>
> **4. Gesture Customization**
> Allow users to remap gestures to:
>
> * Mouse actions
> * Keyboard shortcuts
> * Media controls
> * Custom shortcuts
>
> **5. Performance Panel**
> Show beautiful dashboard cards displaying:
>
> * FPS
> * AI Confidence
> * CPU Usage
> * GPU Usage
> * Camera Resolution
> * Detection Latency
>
> **6. Quick Controls**
> Large modern buttons:
>
> * Start Tracking
> * Pause
> * Calibrate
> * Reset
> * Test Gestures
>
> **7. Settings**
>
> * Camera selection
> * Camera resolution
> * Theme (Dark/Light)
> * Startup options
> * Notifications
> * Auto update
> * Language
>
> ### Visual Style
>
> * Dark theme (#121212 background)
> * Blue and cyan accent colors
> * Rounded cards (16–20 px radius)
> * Frosted glass panels
> * Modern typography similar to Segoe UI or Inter
> * Consistent spacing
> * Elegant icons
> * Soft shadows
> * Smooth gradients
> * Minimal clutter
> * Professional desktop software aesthetic
>
> ### User Experience
>
> The interface should feel like a premium AI productivity tool. Important actions should be highly visible, while advanced settings remain organized and easy to discover. Prioritize usability, clarity, and visual polish.
>
> **Target Platform:** Windows 11 Desktop Application
>
> **Aspect Ratio:** 16:9 (approximately 1400 × 900 px)
>
> Create high-fidelity UI mockups showing the main dashboard, gesture editor, settings page, and calibration screen.

### Even better idea

Instead of making it look like a typical utility, aim for the style of well-known desktop applications:

* **Logitech G HUB** (gaming peripherals)
* **NVIDIA GeForce Experience**
* **SteelSeries GG**
* **Windows 11 Settings**
* **Adobe Creative Cloud**
* **Discord Desktop**
* **Spotify Desktop**

Adding this line to the prompt often improves AI-generated designs:

> **"Use the visual quality and design language of Logitech G HUB, Windows 11 Settings, and Adobe Creative Cloud. The interface should feel premium, futuristic, and commercially ready rather than like a student project."**

This tends to produce cleaner layouts with better spacing, typography, and a more polished overall appearance.
